# Naive-Bayes-Pixel-Level-Image-Segmentation

Implementation of a naive Bayes algorithm for Image segmentation trained with The Oxford-IIIT Pet Dataset.

Dataset Link: https://www.robots.ox.ac.uk/~vgg/data/pets/
